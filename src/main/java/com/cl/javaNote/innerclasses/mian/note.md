  #嵌套类
  内部类声明为static
  1. 要创建嵌套类的对象，并不需要外围类的对象
  2. 不能从嵌套类的对象中访问非静态的外围类的对象
  3. 普通内部类的字段与方法，只能放在类的外部层次上，所以普通的内部类不能有static数据和static字段，也不能包含嵌套类，但是嵌套类可以
