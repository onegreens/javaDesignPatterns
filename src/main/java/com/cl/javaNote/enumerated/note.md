# 枚举类型 
关键字enum可以将一组具名的值的有限集合创建为一种新的类型，而这些具名的值可以作为常规的程序组件使用

# compareTo
用于两个相同数据类型的数据的比较

- 如果指定的数值与参数相等则返回0
- 如果指定的数值小于参数则返回-1
- 如果指定的数值大于参数则返回1
